# CPSC 481 Week 3: Intelligent Agents & Foundations

## Agent Definitions
* **Agent**: Anything that perceives its environment through **sensors** and acts upon it through **actuators**.
* **Percept**: The agent's perceptual inputs at any given instant.
* **Percept Sequence**: The complete history of everything the agent has ever perceived.
* **Agent Function**: A mathematical mapping ($f: P^* -> A$) that maps any given percept sequence to an action.
* **Agent Program**: The actual implementation of the agent function running on the physical **architecture**.

## Rationality
* **Rational Agent**: An entity that "does the right thing" to maximize its **performance measure**.
* **Consequentialism**: Rationality is judged by the outcome (the state) rather than the behavior itself.
* **Omniscience vs. Rationality**: Rationality is about maximizing *expected* success based on what is known; it is not the same as perfection or knowing the future.
* **Autonomy**: An agent is autonomous if its behavior is determined by its own experience rather than just the designer's prior knowledge.

## PEAS Framework
Used to specify the task environment:
* **P**erformance: The objective criteria for success (e.g., safety, speed).
* **E**nvironment: The external world the agent operates in (e.g., roads, weather).
* **A**ctuators: The tools the agent uses to act (e.g., steering, signals).
* **S**ensors: The inputs the agent uses to perceive (e.g., cameras, GPS).

## Environment Properties
* **Observability**: **Fully** (total access) vs. **Partially** (noisy/missing data).
* **Determinism**: **Deterministic** (certain next state) vs. **Stochastic** (uncertainty/probability).
* **Episodic vs. Sequential**: In **Episodic** environments, the current decision does not affect the future; in **Sequential**, history matters.
* **Static vs. Dynamic**: **Static** worlds don't change while the agent thinks; **Dynamic** worlds do.
* **Discrete vs. Continuous**: Refers to the state and time (e.g., Chess is discrete, driving is continuous).

## The Four Basic Agent Types
1. **Simple Reflex**: Acts only on the immediate percept (Condition-Action rules).
2. **Model-based Reflex**: Tracks the world using an internal state to handle partial observability.
3. **Goal-based**: Uses goal information to act purposefully and plan for the future.
4. **Utility-based**: Uses a utility function (happiness) to make rational tradeoffs between conflicting goals.
5. **Learning Agent**: Components include a **Performance Element**, a **Critic** (feedback), a **Learning Element**, and a **Problem Generator** (exploration).
